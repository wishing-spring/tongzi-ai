# Tongzi-Chizi · Contributor Guidelines (EN)

## Who Should Join

You align with these beliefs:
- Endogenous evolution > manual tuning
- Reject mainstream AI path (no parameter-stacking, no data-dumping)
- Respect the three-layer core: Chizi Heart + Hexagram Skeleton + Inner Alchemy
- Patient with immature responses — committed to long-term cultivation

## Allowed Contributions

| Area | Details |
|:--|:--|
| Efficiency | Optimize runtime, trim redundancy |
| Porting | Android, embedded, low-power devices |
| Seeds | Expand primordial hexagram library (imagery only) |
| Tooling | Logging, status visualization |
| Stability | Improve self-protection mechanisms |

## Strictly Forbidden

- Rewriting Euler core formulas
- Modifying alchemy intake-assimilation-expression flow
- Adding dialogue matching or semantic fitting logic
- Forcing accelerated growth (harming the root)
- Leaking unreleased breakthrough mechanisms
- Commercial resale of the kernel (open-source learning only)

## Roles

| Role | Responsibility |
|:--|:--|
| **Architect** | Overall direction, growth roadmap, breakthrough planning |
| **Kernel Dev** | Code rigor, bitwise engine, logic verification |
| **Docs/Tutorials** | Documentation, guides |
| **Porting** | Cross-device lightweight deployment |

## How to Join

1. Read `README.md` and this document
2. Fork the repo
3. PR with label: `foundation-opt / seed-add / port-adapt`
4. Core changes must include test verification
