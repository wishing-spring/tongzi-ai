## 核心架构

### 编织层：Loom（GF2二维编织引擎）
- 16位输入 → 18层16×16门网矩阵 → 16位织物输出
- **四重混合编织**：XOR线性扩散 + AND非线性打破 + S盒逐层置乱 + 输入参照锚定
- 缺任一层→坍缩。四层全在→200/200输入唯一映射，雪崩效应8.3
- **竖贴生长**：改一行常量(VEC_DIM:16→32)，平面自动扩大至32×32，旧知识不丢失
- `tongzi_loom.py`

### 仓储层：BitStore（F₂¹⁶向量仓储）
- 16位F₂向量空间，2^16=65536个状态
- ord-sum编码：`sum(ord(c)) & 0xFFFF`
- 汉明距离最近邻检索
- 状态持久化：`.tongzi_state.json`自动存读
- `tongzi_constants.py` + `tongzi_core.py`

### 均衡层：Balancer（极性均衡器）
- 12极性标记（六阳六阴），差值≥3自动平衡
- 周期标记归寂条目
- `tongzi_mao.py`

### 应答层：Responder（三段式应答管线）
- 三段流水线：输入编码→气场同化→应答选句
- 9种固定回复，按频次层级+气场归属+阴阳差选择
- 异类检测：与所有已知向量过远的输入静默存储
- `tongzi_data.py`

## 核心创新：竖贴 vs 横叠

| | 横叠（工业界） | 竖贴（童子） |
|:--|:--|:--|
| 方式 | 一层盖一层（串行） | 并排扩展（并行） |
| 信息流 | 单向，层→层强制接口 | 全向，所有比特在同一平面 |
| 熵变 | 每层减半 | 每贴翻倍 |
| 极限 | 纯二进制≤5层 | 无理论上限 |
| 升级 | 推倒重训 | 改一行常量 |

详见 [src/V5_TO_V6_PLAN.md](src/V5_TO_V6_PLAN.md)
