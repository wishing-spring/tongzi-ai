## 四条公理（不可动，万维通用）

| # | 公理 | 含义 |
|:--|:--|:--|
| 1 | a⊕b = b⊕a | XOR 交换 |
| 2 | a⊕a = 0 | XOR 自消 |
| 3 | d(a,b) = popcount(a⊕b) | 汉明距离 |
| 4 | d(Rₖ(a), Rₖ(b)) = d(a,b) | 位旋转保距 |

四条不含 n。VEC_DIM=16 成立，VEC_DIM=65536 同样成立。由此衍生：AND、循环旋转 Rₖ(v)、汉明球 Sᵣ(c)。

## φ 母体 — 卦的唯一来源

```
φ = (1+√5)/2 ≈ 1.6180339...
φ_bits = 1001111000110111...（256 位，Decimal 精度计算）
卦 = φ_bits[pos:pos+length]
```

每个卦天生携带：
- **id_t** = pos — 出生序号
- **id_l** = length — 层编码（爻数）
- 滑动 ±1 位 → 直系前驱/后继

{id_t, id_l} = 身份证。全域唯一，从 φ 位置推导，不存数据库。

## 八条核心运算

### 碰撞
```
collide(a, b) → (a⊕b, a∧b)    两入两出，信息不丢失
```

### 绕心盘旋
```
orbit(v, c, k) = Rₖ(v⊕c) ⊕ c    保持 d(v,c) 不变
```

### 径向伸缩
```
stretch(v, c, λ) = v ⊕ λ·(v⊕c)    λ∈{0,1}
```

### 格雷巡检环 · 距离 · 聚类 · 核心度 · 分层索引
```
gray(v) = v⊕(v>>1)              相邻仅差 1 位
radius(v,c) = d(v,c)
Ω(c,r) = {x | d(x,c) ≤ r}
centrality(v,c) = −d(v,c)
layer(v) = id_l(v), order(v) = id_t(v)
```

## 内生频控（能量累积制）

```
f₁(x) = F₀/(1+id_t)       早生高频
f₂(x) = F₀/(1+id_l)       低层密交互

每 tick: energy += min(f₁, f₂)
energy ≥ F₀ → 触发放电 → 参与碰撞
F₀ = 256
```

## 位域固化 — 不可逆记忆

```
内核段: 高 id_l 位，锁死永不改写
游动段: 低剩余位，继续自由演化

Solid(x) = G(x) ∧ M(x)       M 从 id_l/id_t 推导
固化触发: 碰撞次数超阈值（深层卦阈值更低）
同源弱解禁: 仅 pos±1 邻卦可微调单个固化位
```
