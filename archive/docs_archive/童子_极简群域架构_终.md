## 一、本质

童子不是一个算法、不是一个模型、不是一个处理器。

是一个 **F₂ⁿ 空间**。卦是空间里的点。卦与卦之间的关联、匹配、排布、运动——是空间本身的属性，不是外部规则。

```
不是: 输入→编码→处理→输出（流水线）
而是: F₂ⁿ 空间 + 卦群 + 自然交互法则
```

## 二、四条公理（不可动，万维通用）

| # | 公理 | 含义 |
|:--|:--|:--|
| 1 | a⊕b = b⊕a | XOR 交换 |
| 2 | a⊕a = 0 | XOR 自消 |
| 3 | d(a,b) = popcount(a⊕b) | 汉明距离 |
| 4 | d(Rₖ(a),Rₖ(b)) = d(a,b) | 位旋转保距 |

四条不含 n、不含维度、不含任何外部参数。16 爻成立，65536 爻也成立。

衍生基础算子：AND（a∧b ∈ F₂ⁿ）、位循环旋转 Rₖ(v)、汉明球 S_r(c)。

## 三、φ 母体——卦的出生与身份

φ = 1.6180339... 的二进制小数展开是卦的唯一来源。

```
φ_bits = 1001111000110111...

卦 = φ_bits[pos:pos+length]

每个卦天生携带:
  pos    → 出生序号 id_t（第几个从 φ 取出）
  length → 层归属 id_l（取了多少位，2爻/3爻/...）
  滑动1位 → 前驱/后继亲缘
```

**{id_t, id_l} = 卦的身份证。** 全域唯一。不需存储——从 φ 位置即可推导。

## 四、八条核心运算

### 1. 碰撞（信息完全展开）
```
collide(a, b) → (a⊕b, a∧b)
```
输出差异向量和共识向量。两入两出，信息不丢失。

### 2. 绕心盘旋
```
orbit(v, c, k) = Rₖ(v⊕c) ⊕ c
```
v 绕中心 c 在等距球面上旋转。保持 d(v,c) 不变。

### 3. 径向伸缩
```
stretch(v, c, λ) = v ⊕ λ·(v⊕c)
λ∈{0,1}: 0=原位, 1=推至对面
```
改变 v 与中心 c 的距离。λ 逐位异或。

### 4. 格雷巡检环
```
gray(n) = n ⊕ (n>>1)
```
全卦排成环，相邻只差 1 位。天然巡检路径。换维度自动适配。

### 5. 精确距离
```
radius(v, c) = d(v, c)
```

### 6. 批量聚类
```
Ω(c, r) = {x | d(x, c) ≤ r}
```
以 c 为中心、半径 r 的汉明球内全部卦。

### 7. 核心度
```
centrality(v, c) = -d(v, c)
```

### 8. 分层索引
```
layer(v) = id_l(v)          # 从 φ 取位长度
order(v) = id_t(v)          # 在 φ 里的出生序号
prev(v) = φ_bits[pos-1:pos-1+length]   # 前驱
next(v) = φ_bits[pos+1:pos+1+length]   # 后继
```

## 四-B、卦元合体（v1.2 新增）

碰撞瞬间，若两卦游动段汉明距离落在合体窗 (4, 12) 内，**同时产出两条合体子卦**：

### 合体判据
```
h = w_H(a ⊕ b)          # 汉明距离
合体条件:  4 < h < 12   # VEC_DIM/4 ∼ 3·VEC_DIM/4
```
太近（h≤4）→ 无信息增量，不合；太远（h≥12）→ 几乎无关，不合。

### XOR 合体 — 差异关联链
```
C_xor = a ⊕ b           # 两卦差异的结晶
d_H(C_xor, a) = w_H(b)  # 交融深度 = 对方权重
```
信息不丢——C_xor ⊕ a = b，C_xor ⊕ b = a。关联联想、递进推理。

### AND 合体 — 共识沉淀
```
C_and = a ∧ b           # 两卦共识的凝结
```
归纳通用规律、积累共性经验。补上 XOR 丢弃共识信息的先天短板。

### 合体卦生存
- 合体卦为 **临时卦元**，入空间后与原生卦走同一套频控 + 密度规则
- 信息不丢失：原卦 A、B **全程不动**——碰撞行为不变，只是额外产子
- 零值跳过：XOR 或 AND 结果为 0 时产子终止（无信息）
- 密度自清兜底：无用的临时合体卦被现有密度机制自然清理

### 双链语义
| 链 | 运算 | 用途 |
|:--|:--|:--|
| 异链 | A ⊕ B | 联想、对比、递进、反差 |
| 同链 | A ∧ B | 归纳、共识、沉淀、规律 |

## 五、内生频控（非外部定时）

### 层深频控（v1.2 简化）
```
rate(x) = F0 / (1 + id_l)     → 每 tick 能量增量
```
id_l 小 → 层浅 → 速率高 → 频繁碰撞。
id_l 大 → 层深 → 速率低 → 稀交互。
同层卦同速——不因出生顺序 (id_t) 偏斜。

### 能量累积制
```
每 tick: energy += rate
energy ≥ F0 → 触发参与，energy -= F0
```
F0 = 256（基础能量阈值）。固化卦不参与碰撞。合体卦走同一套规则。

## 六、位分区

16 位分四区：

| 位域 | 用途 |
|:--|:--|
| 位 15-12 | 状态码（基底/交互/排布/记忆） |
| 位 11-8 | 参数区（k, λ, mask） |
| 位 7-4 | 身份区（id_l, id_t 压缩） |
| 位 3-0 | 数据区（卦的核心内容） |

状态码 = 自指——卦知道自己处于哪个状态。维度无关——换 n 只需调整各区宽度比例。

## 七、位域固化——不可逆记忆

每个卦天然分两段：

| 段 | 来源 | 特性 |
|:--|:--|:--|
| 内核段 | φ 起始基准位，长度 = id_l 决定 | 根基底色 |
| 游动段 | φ 滑动 pos 位延伸生成 | 后天演化 |

### 固化运算

```
Solid(x) = G(x) ∧ M(x)
M(x) = mask(id_l, id_t)     # 从身份推导，不存数据库
```

M(x) 覆盖的位永久锁定——不参与 XOR/旋转/伸缩。M(x) 外的位继续自由演化。

### 触发条件（内生）

1. **亲缘碰撞达标**：同层序、pos±1 邻卦累计碰撞频次超阈值（id_l 越大上限越低→高层更易固化）
2. **层深自判**：id_l 越大 → 内核占比越大 → 优先固化

### 效果

```
collide(x, y) 只取两方游动段做 XOR/AND
固化段永不丢失、永不改写
```

id_l 小的浅层卦少固多动，id_l 大的深层卦多固少动。

### 同源弱解禁（防僵化）

仅直系前驱/后继（pos±1）可对已固化位做微调。无关卦无法触及核心。

## 八、空间自感知与密度调节

### 局部疏密感知

```
S(x) = N / (d̄_H + 1)
  N = 同层近亲邻域内卦数
  d̄_H = 邻域平均汉明距离
```
S 大 = 拥挤，S 小 = 空旷。

### 联动 λ 伸缩自调节

```
λ_new = λ_base × (1 - μ(id_l) × S/S_max)
```

S 感知到拥挤 → λ 自动增大 → 卦向外散开 → S 回落（负反馈闭环）。

### μ 敏感度——纯内生推导

```
μ(id_l) = id_l / L_max
  L_max = 当空间内最大层编码
```

L_max 不是外部参数——是当前空间内所有卦的 id_l 最大值，运行时自动计算。低层卦 μ 小（耐挤），高层卦 μ 大（一挤就散）。

### 双层控体量

```
轻度拥挤: 自感知散开（事前调节，温和）
极度过载: 密度自清合并（事后兜底，强力）
```
