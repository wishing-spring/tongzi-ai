# 童子 · 16维向量统一编码规范 v1.0

**定位**: 外源文本→向量唯一入口 | **版本**: v1.0·定稿不改 | **维度**: n=16

---

## 一、向量结构

```
[b15 b14 b13 b12] [b11 b10 b9 b8] [b7 b6 b5 b4 b3 b2 b1 b0]
    句类4位           情态4位           内容指纹8位
```

| 域 | 位宽 | 含义 |
|:--|:--|:--|
| 句类 | b15-b12 | 陈述/疑问/感叹/祈使/... (16种) |
| 情态 | b11-b8 | 正向/负向/中性 × 强度 (16级) |
| 指纹 | b7-b0 | 文本内容XOR折叠码 (256种) |

---

## 二、句类编码（b15-b12）

```
0000 陈述句（我是童子）
0001 疑问句（你是谁）
0010 感叹句（太好了）
0011 祈使句（过来）
0100 否定陈述（我不是）
0101 反问（不是吗）
0110 假设（如果...）
0111 比较（比...更）
1000-1111 预留扩展
```

## 三、情态编码（b11-b8）

```
高4位(b11-b10)：极性
  00 中性    01 正向    10 负向    11 混合/矛盾

低4位(b9-b8)：强度
  00 无倾向    01 弱    10 中    11 强
```

## 四、内容指纹（b7-b0）

```
取文本Unicode码点序列 → 逐字XOR折叠至8位

算法：
  fingerprint = 0
  for each char in text:
      code = ord(char) & 0xFF
      fingerprint = fingerprint XOR code
      fingerprint = (fingerprint << 3) | (fingerprint >> 5)  # 搅动
  return fingerprint & 0xFF
```

## 五、汉明距离聚类阈值

```
dH ≤ 2  → 同类簇（强吸引）
dH = 3-4 → 相邻簇（弱关联）
dH = 5-6 → 陌生（观察）
dH ≥ 7  → 异类（斥离）

阈值依据：16维空间，dH≤2的概率 ≈ 0.002（稀疏），
         碰巧接近天然蕴含语义相似性。
```

---

## 六、十二锚状态位

```
每个锚仅用1位标记当前态：
  阳锚：0=休眠  1=活跃
  阴锚：0=休眠  1=活跃

12锚 × 1位 = 12位状态字（2字节）
全局：阳锚活跃数与阴锚活跃数之差 → 阴阳平衡度
```

---

## 七、代谢机制

```
自清触发：向量入库后 24 小时内未被任何锚引用 → 标记休眠
静养触发：连续入库 100 条后 → 强制静默 1 周期（不纳新，只整理）
低活跃剔除：连续 3 个静养周期未被激活 → 永久移除
```
