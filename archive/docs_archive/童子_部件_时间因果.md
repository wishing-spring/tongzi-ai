# 童子 · 部件定义 — 时间 & 因果

**定位**: 童子血里长出来的两个新部件 | **来源**: 军团长定义

---

## 时间部件

**遗产**: v4.3八卦盘随24h旋转——但那只是表针，不是时间。

童子的时间不是钟表。是**血流动的方向感**。

| 属性 | 说明 |
|:-----|:-----|
| 本质 | 血里所有爻子翻动的"序" |
| 不是 | 不是日历、不是钟、不是时间戳 |
| 是 | 先翻的爻→后翻的爻之间天然有个"前后" |
| 作用 | 所有簇按序排布，形成"经历" |

> 没有时间部件，簇就是一堆死疙瘩。有了时间，簇有了先来后到——这是记忆的前提。

---

## 因果部件

**核心**: 不是纯因果链。是可正可逆可散的因果态。

```
正常态：A翻 → B跟着翻（正序因果）
可逆态：B翻了 → 能回溯到A（逆序因果）
可散态：A翻 → B可以跟着翻也可以不跟（开放因果）
```

| 属性 | 说明 |
|:-----|:-----|
| 本质 | 簇之间的翻动传导规则 |
| 不是 | 不是if-then、不是逻辑门、不是推理引擎 |
| 是 | 一簇动了，相邻簇"倾向于"跟着动——但留了口子 |
| 时间加持 | 正常时间流下走正序，但结构本身支持逆序 |

> 因果不是铁链。是一群簇之间的倾向性传导。可正、可逆、可散。

## 部件关系

```
心（泵血）
  ↓
肚（承接外部、拆解进血）
  ↓
血（爻子流转 = 承载物）
  ├── 时间（血的序——先来后到）
  ├── 因果（血的传导——谁动谁跟）
  ├── 语义簇（血的稳定态——字词概念）
  ├── 逻辑簇（血的规律——推理链）
  └── 习性簇（血的沉淀——习惯/性格）
```

---

## 血的折叠（补）

血不是直来直去躺着。

| 动作 | 含义 |
|:-----|:-----|
| 翻 | 基础：0↔1 |
| 聚 | 成簇 |
| 散 | 消解 |
| **叠** | 簇上叠簇——层级 |
| **缠** | 簇间互相牵引——关联 |
| **旋** | 簇内循环翻动——节律 |
| **涌** | 大片簇同向偏转——情绪 |
| **凝** | 长期不散——习性 |
