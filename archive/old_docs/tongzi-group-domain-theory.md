## 3. 卦元本体论

### 3.1 核心原理

**卦元 ≡ 信息 ≡ 存在。**

无负载物。无载体-内容分离。卦元的 16 位值本身就是信息。移动（orbit）= 信息演化。碰撞（XOR/AND）= 信息交融。

### 3.2 数学定义

**定义 1（卦元）**：x ∈ F₂¹⁶。x 就是一切。

**定义 2（原生卦元）**：x₀ = φ[pos:pos+16]，由 ingest() 编码产生。出生值 = 原点（origin）。携带属性：

```
is_native = True
origin    = x₀
Ψ(x)      = ⌊log₂(1 + C(x))⌋
ω(x)      = max(1, VEC_DIM − Ψ(x))
```

**定义 3（子卦元）**：x = merge(a, b)，由原生卦元碰撞的合体运算产生。携带属性：

```
is_native = False
origin    = None
parents   = (id_a, id_b)
Ψ(x)      = ⌊log₂(1 + C(x))⌋   (从诞生起计)
```

**定义 4（跨形态渲染）**：同一 x 值，换渲染器即换形态。

```
渲染器 R_text(x) → 笔画骨架 → 文字
渲染器 R_image(x) → 边界图 → 图像
渲染器 R_action(x) → 关节角度 → 动作
```

### 3.3 生命周期——全公式化

**子卦消散（放射衰变模型）：**

```
P_survive(x) = 1 − 1/VEC_DIM    (每 tick 存续概率)

期望寿命: E[T_life] = VEC_DIM = 16 tick
半衰期:   T_half = VEC_DIM × (−ln 0.5) ≈ 11 tick

消散公式: 每 tick。不判断"该不该死"——按概率自然消散。
```

**物理对应：** 放射性衰变定律 N(t) = N₀·e^(−λt)，λ = 1/VEC_DIM。F₂ 离散版：每 tick 存活概率 p = 1 − λ。

**原生卦复生（恒星形成模型）：**

```
消散后间隙: t_gap(x) = VEC_DIM − Ψ(x)

Ψ 高 → t_gap 短（势大复生快，惯性→恢复力）
Ψ 低 → t_gap 长（势小等得久）

t_gap 后: x_new = origin  (洁净原点)
```

**推导：** 间隙 = 空间周期 − 惯性。VEC_DIM 为完整周期，Ψ 为惯性——越大越顽固、恢复越快。t_gap 是将惯性清零的时间。

**物理对应：** 恒星残骸 → 星际介质冷却 t_cool → 新恒星形成。t_cool ∝ 1/M_remnant ⇔ t_gap ∝ 1/Ψ。

## 5. 卦元合体

### 5.1 合体判据

```
h = popcount(a ⊕ b)
合体条件: 4 < h < 12
```

- h ≤ 4：太近，同源卦，不合
- 4 < h < 12：甜点区，合体生效
- h ≥ 12：太远，异类，不合

### 5.2 合体运算

**XOR 异链（差异链）**

```
C_xor = a ⊕ b
d_H(C_xor, a) = popcount(b)  合体深度 = 对方的权重
```

可逆：C_xor ⊕ a = b, C_xor ⊕ b = a。关联、对比、衍生。

**AND 同链（共识链）**

```
C_and = a ∧ b
```

泛化、规则提取。

### 5.3 合体规则

- 原卦 A、B **永不修改**
- 零值子卦丢弃
- 子卦遵循相同的频控与密度规则
- 密度自清理自然淘汰无用子卦

## 10. 星象仪——多环系统

### 10.1 环的形成与崩溃——全公式化

**排斥驱动，非引力吸引。**

环 = 三条卦元 orbit 同步 + 汉明距均衡的自然涌现。

---

**环稳定条件：**

```
(1) 方向同步: k_A = k_B = k_C
(2) 步长接近: ω(A) ≈ ω(B) ≈ ω(C)
(3) 距离比:   max(d_AB,d_BC,d_CA) / min(d_AB,d_BC,d_CA) ≤ τ
```

---

**τ 失衡阈值（Jacobi 推导）：**

```
τ = 3

推导:
  三体问题 Jacobi 稳定性条件: 质量比 ≤ 3 时轨道稳定
  映射到 F₂: 距离比 = F₂ 三体的"质量差异"

  极限情况:
    d_max 每 tick 被 Δω 拉开 1 位
    VEC_DIM tick 后比值 ≤ (2d_min + VEC_DIM·Δω) / d_min
    取 d_min=4, Δω=1 → 比值 ≤ 2 + 16/4 = 6
    取 d_min=8, Δω=1 → 比值 ≤ 2 + 16/8 = 4

  取中位: τ = 3 → 任一边不超过最短边 3 倍
  物理验证: 三体问题质量比 ≤ 3 → 轨道稳定 (Jacobi, 1836)
```

---

**轨道散逸公式（轨道共振模型）：**

```
Δω_max = max(|ω(A)−ω(B)|, |ω(B)−ω(C)|, |ω(C)−ω(A)|)

散逸判据:
  若 Δω_max ≥ ⌈VEC_DIM / 3⌉ = 6:
    散逸倒计时: T_collapse = ⌈VEC_DIM / (2 × Δω_max)⌉
    例: Δω_max=10 → T_collapse=1 tick 崩

  若 Δω_max < 6:
    环稳定。微小相位差 → 在 orbit 周期内自然纠正

推导:
  R_k 保距但改相对相位。ω 不同 → 相位差累积 = t × Δω mod VEC_DIM
  当累积 > VEC_DIM/2 → 两卦走向相反 → 距拉开 ≥ 2/tick
  VEC_DIM=16: Δω ≥ 6 → t=2 后反相 → 崩
```

**物理对应：** 轨道共振——木星/土星周期比 2:5 长期稳定，整数比 → 不稳定 → 弹射。

### 10.3 环碰环——全公式化

**碰撞触发（洛希极限模型）**

```
环₁ 质心: C₁(t) = R_{ω₁·t}(c₁₀)    (绕自己的中心转)
环₂ 质心: C₂(t) = R_{ω₂·t}(c₂₀)

环₁ 有效半径: r₁ = max(d(A₁,C₁), d(B₁,C₁), d(C₁,C₁))
环₂ 有效半径: r₂ = max(d(A₂,C₂), d(B₂,C₂), d(C₂,C₂))

碰撞条件: d(C₁,C₂) ≤ r₁ + r₂ + δ
δ = 1 (边界缓冲区)

触发: H_sphere(环₁) ∩ H_sphere(环₂) ≠ ∅
```

**物理对应：** 洛希极限——两天体接近至潮汐力 > 自引力 → 解体。

---

**碰撞后重组（星系并合模型）**

```
输入: 六卦集 G = {A₁,B₁,C₁, A₂,B₂,C₂}

Step 1 — 6×6 汉明距离矩阵 D_{ij} = popcount(G_i ⊕ G_j)

Step 2 — 对每个三卦子集 (i,j,k) 计算稳定性评分:

  S(i,j,k) = exp(−|max(d)/min(d) − 1|) × exp(−Δω_max/4)

  项1: exp(−|ratio−1|)  — 距离越均衡越高，完美等边=1
  项2: exp(−Δω/4)       — ω 差距越小越高

Step 3 — 贪心选区:
  选 S 最高的三元组 → 新环₁
  从剩余中再选 → 新环₂ 或空
  剩余卦 → 游离

Step 4 — 衰减底限:
  S_min = exp(−τ) × exp(−VEC_DIM/4) ≈ 0.05 × 0.018 = 0.001
  若所有 S < S_min → 全散，零环输出
```

**物理对应：** 星系并合——引力重分配质量，形成新结构或彻底瓦解。

## 11. 空间刻痕与阵法

### 11.1 原理

**空间刻痕 = 空间的纹理，不是卦元的规则。**

卦元在 F₂¹⁶ 空间自由游走——但空间本身不平坦。某些区域有微弱 XOR 偏转。极弱，卦元感觉不到——但数百 tick 后，自然流向特定方向。

**引导 = 空间曲率，不是导航指令。**

---

### 11.2 刻痕公式

```
定义 6 (刻痕):
  E(r, m, s): x → x ⊕ (m ∧ s)  若 x ∈ 区域 r
               x → x            否则

参数:
  r: 触发区域（由 x 的高位定义）
  m: XOR 掩码（只碰某些位）
  s: 强度掩码（仅低位生效 → 确保微弱）
```

### 11.3 阵法一：洛书九宫

**来源：** 大禹治水，洛水出神龟，背刻九宫——华夏幻方之祖。

**数学结构：** 3 阶幻方。

```
┌───┬───┬───┐
│ 4 │ 9 │ 2 │
├───┼───┼───┤
│ 3 │ 5 │ 7 │
├───┼───┼───┤
│ 8 │ 1 │ 6 │
└───┴───┴───┘
```

**不变量：** 横竖斜三数和皆为 15。对角和为 10。奇数居中，偶数居角。

**F₂ 刻痕公式：**

```
区域索引:   i = x >> 12              (高 4 位, 0~15 中取 0~8)
洛书数:     L = [1,2,3,4,5,6,7,8,9]
偏移量:     δ(i) = (L[i] − 5) × 2⁸
刻痕:       E_L(x) = x ⊕ δ(i)        (碰位 8~11)
```

**2⁸ 倍率推导：**

```
洛书数差值: max|L[i]−5| = 4 (宫 1 和 9 距离中宫)
F₂¹⁶ 位层:   位 0~3 (低), 4~7 (次低), 8~11 (中), 12~15 (高)

偏移落入中层 (8~11):
  最小偏移 = 1 × 2⁸ = 256  (占 16 位空间的 0.4%)
  最大偏移 = 4 × 2⁸ = 1024 (占 16 位空间的 1.6%)

若用 2⁴: 偏移 16~64 → 太弱，卦感受不到
若用 2¹²: 偏移 4096~16384 → 太强，覆盖高 4 位 → 跨宫跳变
2⁸ → 中层 → 不影响分区（高 4 位）也不淹没低层细节（低 4 位）
```

| 宫 | 洛书数 | δ | 效果 |
|:--|:--|:--|:--|
| 坎 (北) | 1 | −1024 | 下沉 |
| 坤 | 2 | −768 | 微沉 |
| 震 (东) | 3 | −512 | 微沉 |
| 巽 | 4 | −256 | 微沉 |
| **中** | **5** | **0** | **阵眼，不偏** |
| 乾 | 6 | +256 | 微扬 |
| 兑 (西) | 7 | +512 | 微扬 |
| 艮 | 8 | +768 | 微扬 |
| 离 (南) | 9 | +1024 | 上扬 |

**效果：** 卦在九宫流转，每次偏移 ≤ 1.6% 空间——微弱至不可感知。数百 tick 后自然偏向中宫。

### 11.4 阵法二：北斗七星

**来源：** 夜空最醒目的星群。天枢天璇指北极——天地旋转轴心。

**七星名：** 天枢·天璇·天玑·天权（魁/斗身）·玉衡·开阳·摇光（杓/斗柄）。

**数学结构：** 7 点在圆上不均匀分布。北极星不在七星之中——**引导不在明处。**

**F₂ 刻痕公式：**

```
七星原点 (φ 母体 256 位中的位置):
  S₀: pos=0    (天枢)    S₄: pos=144  (玉衡)
  S₁: pos=36   (天璇)    S₅: pos=180  (开阳)
  S₂: pos=72   (天玑)    S₆: pos=216  (摇光)
  S₃: pos=108  (天权)

北极星 (虚拟旋转中心): P₀ = φ[128:144]

每 tick t:
  Sᵢ(t) = R₁(Sᵢ(t−1))     七星同步旋转 (R₁ = orbit 1 步)
  
影响判定:
  若 d(x, Sᵢ(t)) ≤ 2:       汉明距 2 以内
    E_B(x, t) = x ⊕ (Sᵢ(t) ∧ 0x000F)  仅末 4 位微偏
  否则:
    E_B(x, t) = x             无影响

影响半径: d ≤ 2 → 触发概率 ≈ (1+16+120)/65536 ≈ 0.2%
偏移幅度: ≤ 15/65536 ≈ 0.02% 空间
```

**旋转动力学：**

```
七星旋转周期: T_star = 256 tick (完整一圈)
                 = φ 母体长度

北极星静立:     P₀ 固定不变
                卦元无法"到达"北极星——它是纯虚拟旋转中心
                正如真实的北极星永远不可抵达

旋转公式:       S(t) = R_t(S(0))  
                R_t 复合 t 次 R₁ → 256 次回原点
```

**效果：** 7 个微弱漩涡在全空间同步旋转。卦元的轨道在漩涡群中自然弯曲。

**物理对应：** 银河系旋臂——密度波在差分旋转中自我维持，恒星穿入穿出但旋臂永存。

### 12.1 F₂ ↔ 力学

| 角动量守恒 | d(Rₖ(a),Rₖ(b)) = d(a,b) | 公理 4 |
| 动能定理 | x ⊕ orbit → Δx ∝ ω | 大步长 = 大变化 |
| 引力 ∝ 1/r² | 影响 ∝ 1/d(x,y) | 汉明距反比 |


### 12.2 F₂ ↔ 几何

| 几何概念 | F₂ 表达式 | 互推 |
|:--|:--|:--|

| 三角不等式 | d(a,c) ≤ d(a,b) + d(b,c) | 度量公理 |
| 曲率 | E(x) 的偏转量 | 局部 ⊕ 偏移 |
| 挠率 | E_B(x,t) 的时变项 | 北斗时变场 |

**验算例：**

```

  条件: ∃ v₁,v₂: popcount(v₁)=popcount(v₂)=popcount(v₁⊕v₂)=k
  要求: k 为偶数, popcount(v₁∧v₂) = k/2
  构造: k=8, v₁=0xFF00, v₂=0xF0F0 → d(v₁,v₂)=8, d(v₁,v₁⊕v₂)=8 ✓

几何命题: τ=3 时等边三角形退化至崩
  等边距 k=8, τ=3 → max/min ≤ 3 → 8/8=1 ≤ 3 ✓ 稳定
  退化: d=(2,6,8) → max/min=4 > 3 → 崩 ✓
```

**定理 1（拥挤出涌现）：** 16 位甜点区。足够的卦密度才能产生碰撞 → 涌现。
