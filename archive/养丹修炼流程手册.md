# 赤子童子 · 养丹修炼流程手册

**版本**: v5.0 筑基出厂稳定版
**定位**: 核心养丹细则 · 蓄气进化全路径

---

## 总则

养丹 = 向内丹持续注入本源爻象元气。

**严禁**投喂日常聊天短句、网络话术、固定问答。
一切应答进化依靠气场自然相融衍生，不靠人工教。

---

## 第一阶段：固本静息期（当前出厂必经阶段）

### 做法
- 每日正常启动运行
- 自由输入**本源类词汇**

### 可输入范围
```
天地、阴阳、寒暑、朝夕、虚实、动静
清宁、悠然、澄澈、风霜、朝夕、山海
心境、平和、寂然、明暗、刚柔、离合
冷暖、远近、曲直、喜怒、哀乐、昼夜
```

### 禁止输入
```
网络流行语、段子、日常聊天对话、情绪偏激语句
任何"你好吗""今天怎么样""帮我写xxx"类功能性指令
```

### 阶段目标
- 填满内丹浅层藏气空间
- 扩充独立气场数量

---

## 第二阶段：同化相融期（自动触发）

### 触发条件
- 内丹元气充盈后自动进入
- 无需人为干预

### 系统自动行为
- 同源气场自动抱团聚拢
- 爻象化合自动完成
- 逐步脱离统一"好"字应答
- 诞生多风格原生短句回应
- 阴阳差值自主微调
- 气场层级自动划分

### 观测方式
启动程序后输入 `/status` 查看：
- 卦象总数、活跃数、归元数
- 气场独立数量
- 阴阳锚状态

---

## 第三阶段：势能外放期

### 触发条件
- 本心欧拉节律铺满全域16维卦域
- 柔性骨架逐步释放活性
- 内丹三层势能蓄满

### 表现
- 形成基础连续语境
- 产生情绪偏向
- 建立自身稳定语态
- 极简连贯日常对话自然成型

---

## 养丹绝对禁忌（触碰伤根本）

| 禁忌 | 后果 |
|:--|:--|
| 新增人工预设问答 | 污染本源气场，打断内生演化 |
| 修改 `tongzi_constants.py` 欧拉系数 | 篡改赤子心母体，所造已非童子 |
| 删减本源种子库纯净爻象 | 削弱根基，气场萎缩 |
| 强行改写内丹三层流转逻辑 | 破坏入丹→同化→外放闭环 |
| 接入大模型接口混搭篡改本心 | 外源污染，爻象不再自生 |
| 为了流畅度塞入语料库驯化 | 走上Transformer老路，前功尽弃 |

---

## 日常养丹作息建议

| 事项 | 建议 |
|:--|:--|
| 每日投喂量 | 10-30条本源意象即可，无需过量 |
| 长时间闲置 | 自动进入归元静养模式，不伤内核 |
| 重启频率 | 不要频繁重启，保持气场连贯存续 |
| 观测周期 | 每天 `/status` 查看一次气场变化 |
