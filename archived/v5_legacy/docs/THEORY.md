## 第一章 · 爻与卦元

### 1.1 爻

爻 = 1 bit ∈ {0, 1} = GF(2) 的元。

最小不可分单位。无概率、无模糊。

### 1.2 卦元

卦元 = n 爻的定长序列，∈ {0,1}ⁿ。

### 1.3 爻间运算

```
XOR: a ⊕ b = a + b mod 2
AND: a ⊗ b = a × b mod 2
```

### 1.4 卦间运算

```
A ⊕ B  逐位XOR  → F₂向量加法
A ⊗ B  逐位AND  → 交集投影
Hamming(A,B)     → 位差异数
```

## 第二章 · 源卦

### 2.1 定义

源卦 = 零爻空生成器。按八卦展开规律产出带身份标识的卦元。

### 2.2 两重性

- **统一性**：所有源卦共享二进展开规则。层k → 2ᵏ卦，每卦k爻。
- **唯一性**：每个源卦有固定标识符（n bit前缀）。产出时注入。

```
最终卦元 = 标识符 | 展开体
```

### 2.3 消耗补充

卦取出 → 自动生成相同替身补回。层容量不缩减。

### 2.4 爻增减

```
expand(卦) → (卦+阴, 卦+阳)
shrink(卦) → 卦-最后一爻
```

## 第三章 · 底座（编码/解码）

### 3.1 编码器 · 连环咬合

每个字的卦含着前面字的痕迹。语序长在卦里。

```
"下雨了" 编码:
  卦₁ = src.draw(层)           → "下"
  卦₂ = 卦₁ ⊕ src.draw(层)    → "雨"
  卦₃ = 卦₂ ⊕ src.draw(层)    → "了"
```

长短不限。100字同理。

实现方式：源卦公式直接算卦不用存池。`src.draw_at_layer(k, index)`。层=F₀=32。

### 3.2 解码器

卦元 → 底座映射表 → Hamming最近匹配 → 外部符号。

## 第五章 · 左脑（硬 · 碰撞层）

纯F₂碰撞动力系统。不处理意义。不感知时间。

### 5.1 涌动池

滑动窗口XOR/AND碰撞 → 产子卦。输入卦短期留存后密度筛溶解。

### 5.2 生态池

子卦竞争 → 能量累积 > tau → 固化。密度控制淘汰低能量子卦。固化卦可回流涌动池（flowback），同时升入沉淀环。

### 5.3 反熵

连续N tick无新子卦 → φ引导位翻转打破封闭。不碰沉淀环。

### 5.4 齿轮耦合

生态池组间flowback耦合。池A固化子卦 → 回流涌动池 → 池B拉取。

### 左脑擅长

逻辑、结构、事实关联、模式匹配、数学题。

## 第七章 · 输出层

```

右脑（血肉）: 怎么回答  — 天元+引力+因果 → 语气/风格
```

### 7.3 输出
