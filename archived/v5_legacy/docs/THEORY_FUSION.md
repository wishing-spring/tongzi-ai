## 第二章 · 为什么两条河不合并

### 2.1 材质不同

```
童子的世界: 位。只有0和1。距离=Hamming。
灵犀的世界: 位点。每个位点带着语义。距离=靠近谁。
```

合并任一方都会毁掉该方的核心能力。

### 2.2 互补关系

```
童子的独有能力 ←→ 灵犀的独有能力

  自组织生长          ←→   语义理解
  碰撞涌现结构        ←→   规则匹配
  沉淀环积累          ←→   脊骨峰谷
  反熵打破僵化        ←→   因果链偏转
  零训练·纯F₂         ←→   传统卦文化·时间感知

  童子没有的:             灵犀没有的:
  语义·情绪·时间感       生长·涌现·新结构
```

### 2.3 分工不混合

```
童子做底层: 碰撞产生结构。
灵犀做上层: 语义赋予意义。

桥不做翻译。桥做投影。
同一块盘上——童子看它是位模式，灵犀看它是语义位点。
```

## 第三章 · 桥 · 投影

### 3.1 八个参考码

源卦₀/层3产8卦(2³)。自然对应八卦。

```
乾 000: 0b000_000
兑 001: 0b000_001
离 010: 0b000_010
震 011: 0b000_011
巽 100: 0b000_100
坎 101: 0b000_101
艮 110: 0b000_110
坤 111: 0b000_111

格式: 4位标识符(0000) + 3位展开体
取后28位: 标识符占4位+dummy25位+3位展开体 = 低3位区分八种
```

### 3.2 投影算法

```
童子吸引子(28bit)
    ↓
对8个参考码逐一算Hamming距离
    ↓
取距离最小的卦
    ↓
返回: 卦名 + 卦的语义标注
    ↓
送入灵犀的八卦盘上对应位置
```

**不做的事**：
- 不改变童子位值
- 不改变灵犀向量
- 不缓存中间状态
- 不在桥内做任何融合运算

### 3.3 为什么是Hamming而不是别的

Hamming距离天然对应童子底层的XOR运算。两个卦XOR后的二进制中1的个数 = Hamming距离。同底运算，不引入新度量。

```
距离(吸引子, 参考码) = popcount(吸引子 ⊕ 参考码)
```

popcount = 位运算，硬件级速度。

## 第四章 · 并行运行机制

### 4.1 时间驱动

两系统共享同一个时钟。每tick：

```
tick:
  童子: 涌动弹窗 → 生态生子 → 反熵检测
  灵犀: 盘旋转 → 折叠心跳 → 引力拉动
```

不是童子跑完灵犀再跑。是同一tick、并行推进。

### 4.2 输入分发

用户输入同时进入两系统：

```
"下雨了"
    ↓
┌───────────┴───────────┐
↓                       ↓
童子编码器              灵犀字符投影
咬合XOR → 卦₁卦₂卦₃    每个字→盘上投卦
↓                       ↓
涌动弹窗碰撞            规则树匹配
↓                       ↓
生态池固化              世界层积累
↓                       ↓
吸引子(28bit)           规则向量(8D)
    ↓                       ↓
    └───────┬───────────────┘
            ↓
      桥: Hamming投影
            ↓
      灵犀盘上位置
            ↓
      天元·折叠·引力·因果
            ↓
         输出
```

### 4.3 灵犀收童子的东西

灵犀收到的不只是童子投影过来的卦名。还收到：

| 来自童子 | 灵犀怎么用 |
|:--|:--|
| 吸引子投影卦 | 落到盘上→影响天元漂移方向 |
| 沉淀环检索命中 | 作为"经验权重"注入世界层 |
| 碰撞活跃度 | 影响因果张力的计算 |
| 反熵触发 | 作为"僵化预警"传给第九卦 |

灵犀的因果链五流中，其中两流来自童子：
- 字符卦象 ← 童子编码器产出的咬合链的分布特征
- 经验权重 ← 童子沉淀环的检索命中

## 第六章 · 八卦语义锚点

### 6.1 八字出厂标注

每个卦不是只有位模式。有一个语义锚点——告诉灵犀这个卦"代表什么"。

| 卦 | F₂参考码(28bit低3位) | 语义 | 情绪 | 行为倾向 |
|:--|:--|:--|:--|:--|
| 乾 000 | 000 | 天·刚健 | 坚定 | 直接·主导 |
| 兑 001 | 001 | 泽·愉悦 | 开心 | 分享·交流 |
| 离 010 | 010 | 火·急躁 | 激动 | 行动·冲突 |
| 震 011 | 011 | 雷·动荡 | 不安 | 变化·突破 |
| 巽 100 | 100 | 风·渗透 | 柔和 | 适应·迂回 |
| 坎 101 | 101 | 水·危险 | 忧惧 | 防御·谨慎 |
| 艮 110 | 110 | 山·静止 | 沉稳 | 坚持·不动 |
| 坤 111 | 111 | 土·包容 | 平和 | 接纳·滋养 |

### 6.2 语义不参与F₂运算

这些语义标注只在灵犀侧使用。童子侧只看到28位位模式。童子不知道"坎"是水是危险——它只知道Hamming距离最近的是101。

语义和位模式**同一块盘上的两个视角**。童子看位、灵犀看义。互不污染。
