# 16根柱子 + 3台弹珠机 = 83%语义准确率，零训练

## 一句话

**我们发明了一种新编码——弹珠机编码，不训练、不调参、纯位运算，31个汉字的语义判断准确率打到83%。**

---

## 怎么来的

胖子（军团长）问："你玩过电玩城弹珠机吗？铁球掉下来撞柱子，不同球走不同路。"

铜须（铁匠）："懂了。"

然后花了不到一杯茶的时间，把 16 根 XOR 柱子写进了编码机。三台不同柱阵的弹珠机同时跑——三个独立判断，交汇锁定。

---

## 核心技术

```
码点 → 逐位撞16根XOR柱 → 32位向量
     × 3台不同弹珠机 → 三轴交汇
     → 内生因果机 → 归约/剪枝
```

**铁律**：零浮点 · 零矩阵 · 零梯度 · 零词嵌入 · 零注意力 · 零自回归
