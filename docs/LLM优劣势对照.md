## 一、LLM的墙（他们绝对舍弃不了的东西）

| 壁垒 | 具体 |
|:-----|:-----|
| 算力 | 数千GPU·月，天价资金电力 |
| 数据 | 海量全网预训练，TB级文本 |
| 体积 | 几GB~几十GB，无法微型化 |
| 概率 | 统计next-token，无自我存续 |
| 网络 | 完整版必须在线，隐私外流 |
| 千人一面 | 所有人用同一个模型基底 |

**核心矛盾**: 大厂靠堆算力堆数据取胜，这条路他们永远不会放弃。
