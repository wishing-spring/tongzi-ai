## 三件套数理体系

### 1. 意象偏序格 — 承载语义层级

```
意象集合 S
偏序关系 ≼（非全序、可分叉、可回溯）
偏序格 L(S, ≼)

不是固定层级。思绪沿偏序自然爬升、跳转、回溯。
→ 对应人类顺向思考 + 反向复盘
```

### 2. 潜势场 — 流转动力

```
语义潜势场 Φ(x)
无固定数值。只有势能高低差。
意象从高势 → 低势自动流动聚合。
关联紧密 → 势能自发靠拢 → 语义脉络自串联。
不设权重。势能自发消长。
```

### 3. 柔性邻域拓扑 — 生发规则

```
软边界邻域（非硬边界）
邻域内自由跨界融合
边界交叠处 → 自发诞生全新中间意象
→ 不依赖梯度/迭代。靠拓扑间隙融合完成生长。
```
