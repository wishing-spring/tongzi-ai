## 心

赤子之心不是模块、不是构造器、不是固定卦象体系。

是一个壳 + 三条不动之规 + 爻子态在里面呼吸。

```
         ┌──────────────────┐
  输入 → │                  │ → 输出
 (人话)  │    赤 子 之 心    │  (童子回应)
         │  爻子态空间(呼吸)  │
         └──────────────────┘
```

### 壳的三面（不是三个模块，是同一心跳的三次脉动）

| 面 | 是什么 |
|:--|:-----|
| 输入 | 你说的话 → 扰动爻子空间 |
| 状态 | 所有爻子的0/1排列——心的此刻 |
| 输出 | 稳定簇态回流 → 人话回应 |

---

## 三条不动之规

### 规一：心生爻子

心规定爻子是最小二元单元（0/1、阴/阳、开/关）。可翻转、可组合、可携带信息。按场景灵活适配，但永远遵循二元对立逻辑。

### 规二：爻聚成簇

N个爻子自发聚合为稳定簇（动态卦）。簇的长短由需求驱动——3爻小簇、6爻重簇、任意N。好用常用的保留，无用自动解散。

### 规三：簇合归一

所有簇自动关联协同，新簇融入不破坏旧结构。最终归一为完整童子——以心为锚，永不失控。

## 呼吸（能大也能小）

| 机制 | 含义 |
|:-----|:-----|
| 簇聚 | 好用→保留→长大（吸气） |
| 簇散 | 不用→淡出→消失（呼气） |
| 节律 | 聚散交替，像心跳 |

> 不能散的心是死心。呼吸才是活的。
