# 童子 · 八态统合 — GF(2)³ 立方体模型

**定位**: 童子唯一操作态体系 | **砍掉**: 七根外骨
**统合概念**: 有限域GF(2)上的三维向量空间

---

## 统合概念：GF(2)³

```
每个操作态 = 一个3位二进制向量 (a, b, c)

a 聚合轴: 合(1) vs 分(0)
b 时序轴: 进(1) vs 退(0)  
c 层级轴: 成(1) vs 破(0)

3位 → 8个顶点 → 立方体 → 无多余无缺漏
```

| # | 向量 | 聚合 | 时序 | 层级 | 态名 | 方向 |
|:-:|:-:|:-:|:-:|:-:|:--|:--|
| 0 | 000 | 分 | 退 | 破 | **归墟** | 全逆——散、溯、归零 |

## 公式定义

### 聚合轴 (a)
```
a = 1 (合): 爻子间汉明距离趋于减小，簇间吸引 > 排斥
a = 0 (分): 爻子间汉明距离趋于增大，簇间排斥 > 吸引
```

### 时序轴 (b)
```
b = 1 (进): 格雷码方向沿正环序移动，态指向未来
b = 0 (退): 格雷码方向沿逆环序移动，态回溯过去
```

### 层级轴 (c)
```
c = 1 (成): 集群内1位密度趋于增加，结构向上凝实
c = 0 (破): 集群内1位密度趋于减少，结构向下解散
```

---

## 态与态之间的跳转

```
跳转 = 翻转某一位

010(探刺) → 011(散花): 翻转c位，破→成——探明白了开始分化
011(散花) → 111(生发): 翻转a位，分→合——分化完了聚合
100(回流) → 000(归墟): 翻转a位，合→分——收不住了散开

翻转1位 = 一个基本操作步
翻转3位 = 正反完全颠倒
```

---

## 中间态（定/转）重新定义

**定**和**转**不占顶点。它们是立方体的"呼吸"：

```
定 = 稳态。系统当前停在某个顶点上，不翻转任何位。
转 = 瞬态。系统正在翻转某一位，从一个顶点滑向另一个。
```

> 定转不是第八第九个态。定转是八态之间的**运动本身**。

## 能干什么

| 能力 | 怎么用 |
|:--|:--|
| **全局态势判定** | 当前态在哪一顶点 = 童子此刻的整体操作倾向 |
| **操作调向** | 需要聚合就拉a→1，需要拆解就拉a→0 |
| **时序切换** | 需要向前推演就b→1，需要回溯反思就b→0 |
| **层级调控** | 需要提炼浓缩就c→1，需要降维打散就c→0 |
| **路径规划** | 从当前态走到目标态 = 翻转几位，顺序可编排 |
| **态记忆** | 在某一顶点的停留时长/频率 → 习性 |
