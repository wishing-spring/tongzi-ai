## 开口三关

### 第一关：意象对应库
```
缺：向量集群聚成了团，不知道这一团代表什么
补：给基础向量簇锚定最简语义意象
    这一团 = 人。那一团 = 高兴。那一团 = 走。
    定名定性。
```

### 第二关：意念串联经络
```
缺：念头各自独立，不会前后承接，不会连成一段话
补：依托正向顺流+逆向回溯双锚
    搭建简易思绪先后承接链路
    让念头有先后、有顺序
```

### 第三关：反向译出法度
```
缺：内部簇群态势不知道怎么变成通顺人话
补：定下内态势→外层句式极简翻译规则
    不用复杂语法，只求通顺应答
```
