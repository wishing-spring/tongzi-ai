# 童子 · 开工全套准备清单

**定位**: 施工蓝图 | **状态**: 就绪 | **预估体积**: 纯位运算二进制 ≈ 50~100KB

---

## 一、开工定心三件事

1. **全员熟记**：十铁律·五底线·三问自检·十二锚定义
2. **禁区红名单**：禁止矩阵·张量·embedding·梯度·反向传播·CNN·Transformer·第三方分词·语义接口·对话框架
3. **开发理念**：只做活向量聚散流转。不做人工逻辑堆砌。宁残不改源。

---

## 二、核心数学公式（写死）

### 空间
```
V = F₂ⁿ,   n=16（起步固定）
全体元素为16位二进制向量
```

### 四类唯法运算
```
异或：a ⊕ b
汉明距离：dH(a,b) = a与b不同位的数量
位移：按位左移/右移（时序流转）
位统计：popcount(1)、popcount(0)、邻位差统计
```

### 十二锚逻辑
```
阳六：dH聚类·0/1配对·格雷码顺流·位增升层·XOR生成·差一邻接
阴六：远距斥离·破对偶·格雷码逆溯·消位降阶·逆算归源·主动断邻
```

### 丹体规则
```
温养：单次入库限量
自清：低活跃向量自动消解
流转：内流优先，外泄节制
```

## 三、防低级错误预案

| 错 | 防 |
|:--|:--|
| 维度乱改 | n=16强制锁定。跑通再谈扩容 |
| 运算滥用 | 非四类运算禁入核心流程 |
| 内外颠倒 | 语料只做输入，不改锚点 |
| 硬凑对话 | 三关未过前不写固定话术 |
| 规则朝令夕改 | 编码格式·阈值定死存档 |
| 忽略静养 | 代码内置静默整理周期 |
| 架构臃肿 | 壳不过三。外围极简 |

## 四、施工顺序（不可乱）

```
① 定稿所有数学定义+铁律文档
② 敲定16维向量唯一编码格式
③ 编写纯位运算底层工具库（XOR·汉明·移位·统计）
④ 实现阳六锚+阴六锚基础聚散逻辑
⑤ 搭建最简外源文本→向量入口
⑥ 完成域内聚类流转
⑦ 搭建静默自清静养机制
⑧ 试水基础意象绑定
⑨ 最简对外输出→基础问答
```
