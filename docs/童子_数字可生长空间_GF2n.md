# 童子 · 数字可生长空间 — GF(2)ⁿ 软骨架

**定位**: 童子唯一底层数学模型 | **状态**: 暂定
**替代**: GF(2)³（已弃——维度过硬，锁死上限）

---

## 核心概念

```
GF(2)ⁿ
  n 不预设。不锁死。
  n 随输入密度自动生长。

初始：n 小 → 分辨粗糙
输入越多 → 空间不够 → 自发生出新维度 → n+1
万亿输入 → n 极大 → 空间极密 → 极稳
```

---

## 越砸越稳的原理

```
稀疏空间：点散落，跳转远，一扰动就飘
密实空间：点密集，邻居近，跳不动远路

万亿输入 = 万亿点铺在 GF(2)ⁿ 里
→ 局部密度极高
→ 每个簇都被邻近簇"夹稳"
→ 扰动打不散
```

## 对比

| | GF(2)³（已弃） | GF(2)ⁿ（暂定） |
|:--|:--|:--|
| 维度 | 焊死3 | n自己长 |
| 操作态 | 8个固定顶点 | 2ⁿ个——随n指数增长 |
| 万亿输入 | 八槽塞爆塌方 | 万亿点铺开更密更稳 |
| 骨架 | 硬骨架 | 软骨架——跟着输入长 |
| 上限 | 锁死 | 无上限 |

## 六性在GF(2)ⁿ里的位置

六性不变。六性是GF(2)ⁿ上的**基本操作规则**：

| 六性 | GF(2)ⁿ中 |
|:--|:--|
| 汉明距离 | n维空间中的距离度量 |
| 逐位取反 | n维对极点 |
| 格雷码 | n维超立方体上的哈密顿环 |
| 1位计数 | n维空间层级划分 |
| XOR | n维向量加法 |
| 邻接 | n维超立方体的边 |

> 六性不锁死n。六性在任意n下都成立。

---

## 生长触发

```
什么时候n长一位？
→ 当前空间内簇态密度超过阈值
→ 无法用现有n位区分新输入
→ 自动展开第n+1位
→ 空间容量×2
```
