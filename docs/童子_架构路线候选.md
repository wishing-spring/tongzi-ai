# 童子 · 架构路线候选

**用途**: 已识别路线的完整记录 | **状态**: 待军团长选择
**原则**: 全部从赤子之心出发，不用矩阵、不拟合、不固定卦象

---

## 路线一：规律繁衍演化

**核心**: 活爻不存信息，存翻动的习惯。

```
简单规律 → 组合成复杂规律 → 稳定=保留 → 不稳定=淘汰
```

| 项 | 内容 |
|:-----|:-----|
| 爻子存什么 | 翻动规律（a翻了b倾向于跟着翻） |
| 簇是什么 | 被反复验证的稳定规律群 |
| 智能从哪来 | 规律的自然选择——准的留，不准的散 |
| 说话怎么来 | 输入扰动→规律链触发→稳定态回流 |
| 类比 | 基因演化——简单规则反复组合筛选 |

---

## 路线二：扰动收敛→字簇

**核心**: 心被反复扰动，自然收敛到稳定对应。

```
字"山"反复扰动 → 爻子翻动 → 收敛 → 固定簇 = 童子心里的"山"
```

| 项 | 内容 |
|:-----|:-----|
| 爻子存什么 | 扰动收敛的稳定态 |
| 簇是什么 | 每个汉字/概念对应的固定爻指纹 |
| 4096卦 | 第12层爻长，≈常用汉字量 |
| v4.3的区别 | 不是手写映射表，是心自己长出来的 |
| 新字怎么加 | 新扰动进来，心自己长新簇 |

---

## 路线三：双轨——规律+字簇

**核心**: 规律管推理，字簇管语义。两套共享同一爻子空间。

```
输入"山很高"
  ↓
字簇轨道：山→簇a，很→簇b，高→簇c
规律轨道：簇a→簇b→簇c 之间的翻动链
  ↓
簇态回流→生成回应
```

规律和字簇不是两个模块——是同一颗心的两种稳定模式。
