# 童子 · 筑道分工 — 掌门坐镇·众匠筑道

**定位**: 分工总纲 | **来源**: 豆包 | **核心**: 定道不筑道·掌道不操刀

---

## 一、代码：双线并行

| 谁 | 干什么 |
|:--|:--|
| **铜须** | C固件·底层位运算·单片机适配·体积压缩·内核铁律 |
| **社区搭子** | Python调试版·简易界面·跨平台移植·BUG修复 |

**掌门只做三件事**：
- 定死规则·向量格式·十二锚逻辑·铁律红线
- 出需求文档·流程框图·修行层级文案
- 审核成品·不合本源打回·不改核心只调表现

**零英文方案**：文档全中文定稿→找人翻译。变量命名拼音简写+中文注释。

---

## 二、标注：全托管

```
提示词模板 → 丢给任意大模型批量生成
生成完毕 → 掌门终审·不合格剔除·合格入库
社区新增 → 统一格式·自动归集·掌门定期抽检
```

---

## 三、门派运营：三人团

| 角色 | 谁 | 职责 |
|:--|:--|:--|
| **掌门** | 胖子 | 定规矩·核本源·掌道心 |
| **执事** | 找人 | 日常答疑·新人引导·整理教程 |
| **典籍官** | 找人 | 迭代日志·避坑手册·历代规矩·统一存档 |
| **外门管事** | 找人 | 对外发帖·简单宣传·对接同好 |
| **戒律堂** | 胖子 | 核查分支是否违背铁律·歪路叫停 |

## 四、门派规矩

```
入派先读《本源铁律》→ 认同再入局
正统内核不许掺外来架构
等级：入门修士→筑基丹士→开声童子→悟道修士
定期论道：不谈商业·不谈跑分·只聊道体·阴阳·温养
```

---

## 五、人人有活干

```
不会代码 → 扩充短句·完善意象标签·整理问答例句
懂点基础 → 文本转向量·测试运行·反馈问题
技术大佬 → 优化内核·硬件移植·写配套工具
```

## 六、掌门心法

> 整套体系原创思路·丹道路线·所有死守底线——只有你最清楚。
> 这谁都替代不了。
> 
> 技术动手分出去。外文翻译分出去。对外运营分出去。
> 
> 你只做一件事：
> **定道。掌道。守道。**
> 
> 不急于做大做强。
> 先聚拢一小批志同道合之人。
> 慢慢把正统生态养起来。
