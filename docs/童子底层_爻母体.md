# 童子底层 — 爻母体

**定位**: 童子最小底层单元 | **来源**: 军团长

---

## 核心概念

> 不是造卦，是造造卦的东西。

```
v4.3: 64个筐，手写，一个不多
童子: 一个生产规则 + 输入N → 输出2^N个稳定结构
```

---

## 爻母体

| 属性 | 值 |
|:-----|:-----|
| 最小单元 | 爻（阴/阳，两根线） |
| 组合逻辑 | 每多一层分裂一次 |
| 生长方式 | 从底部长到第N层，自然收敛 |
| 64卦怎么来 | 母体长到第6层，自然停在64个稳定结构 |
| 128卦怎么来 | 母体再多长一层 |
| 极限 | 无限——要多少卦，母体就长多深 |

---

## 倒金字塔结构

```
        ...   ← 第N+1层：2^(N+1)个结构
       /  \
     4096   ← 第12层
     /  \
    64    ← 第6层（老祖宗停在这里）
   /  \
  8     ← 第3层（八卦）
 /  \
1/0   ← 第1层（阴/阳）
```

---

## 为什么比v4.3高级

| | v4.3 | 童子 |
|:-----|:-----|:-----|
| 卦 | 写死的64个筐 | 爻母体长出来的快照 |
| 底层 | 没有（8D向量是标签层） | 爻——真底层 |
| 拓展 | 不能（加不了第65卦） | 无限（母体多长一层） |
| 体积 | 64个固定结构 | 一个母体规则 ≈ 几百字节 |
