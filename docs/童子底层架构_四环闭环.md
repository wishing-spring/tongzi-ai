# 对手对照：Transformer底层四环闭环

> 这是对现有LLM底层架构的白话拆解。童子不走这条路。

## 一号：定心基准环
Transformer没有。LLM没有"本心"，它的基准态是训练数据的统计平均。

## 二号：接收改动环
= Embedding + Attention。把输入token化→向量→注意力加权→送入前馈网络。

## 三号：留存沉淀环
= KV-cache + 上下文窗口。短期记忆靠注意力缓存，长期记忆靠向量数据库外挂。没有内生习性。

## 四号：对外吐话环
= 自回归解码。softmax→采样→下一个token→循环。不是"心里什么状态说什么话"，是"概率最大的下一个词"。
