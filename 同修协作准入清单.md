# 赤子童子 · 同修协作准入清单

**版本**: v5.0 筑基出厂稳定版
**定位**: 共建人员规矩 · 准入标准 · 分工边界

---

## 一、准入三观要求

认同以下核心理念者方可入伙共建：

1. **内生演化大于人工调教** —— 不推崇堆参数、灌数据训练模式
2. **拒绝主流AI套路** —— 不走Transformer、不用预训练、不加注意力机制
3. **尊重三层核心架构** —— 赤子心+内丹+柔性卦骨，不推翻底层设计
4. **能接受短期稚拙应答** —— 耐心长期养丹成长，不追求立竿见影

---

## 二、允许做的事（合规开发）

| 方向 | 具体内容 |
|:--|:--|
| 效率优化 | 优化底层运行效率、精简冗余运算 |
| 跨端移植 | 适配安卓/嵌入式/低端设备 |
| 种子扩充 | 扩充纯净本源种子爻象（仅限意象类） |
| 观测工具 | 完善运行日志、状态可视化面板 |
| 稳定性 | 优化自保平衡机制，提升运行稳定性 |
| 气场标注 | 整理不同气场分类标注，辅助养丹 |

---

## 三、绝对禁止触碰（铁律）

| 禁止事项 | 严重性 |
|:--|:--|
| 重构欧拉赤子心母体核心公式 | ⛔ 毁本心 |
| 改写内丹入丹-同化-外放三层通路 | ⛔ 断运化 |
| 添加对话匹配、语义拟合类逻辑 | ⛔ 走老路 |
| 强行加快生长速度揠苗助长 | ⛔ 伤根基 |
| 私自公开未放出的破格进阶底牌 | ⛔ 泄门规 |
| 商用倒卖底层内核 | ⛔ 违开源 |

---

## 四、共建分工划分

| 角色 | 负责 | 当前人员 |
|:--|:--|:--|
| **架构主控** | 把控底层走向、成长路线、底牌规划 | 胖子（军团长） |
| **底层硬核优化** | 代码严谨性、底层逻辑校验 | 铜须 |
| **文档/科普/教程** | 统一整理完善使用说明 | （待招募） |
| **移植适配** | 多设备轻量化部署 | （待招募） |
| **标注/运营策略** | 气场分类、社区运营 | 豆包 |

---

## 五、加入方式

1. 阅读 `README.md` 完整架构说明
2. 阅读 `养丹修炼流程手册.md` 理解养丹逻辑
3. Fork 仓库，提交 PR 时标注：`筑基优化 / 种子增补 / 适配优化`
4. 底层改动需附测试验证结果

---

## 六、底线声明

本项目不追求**用户数量**、不追求**交互流畅度**、不追求**大众口碑**。

只追求：**本源纯净 · 内生演化 · 架构严谨 · 轻量极致**。

认同则来。不认同请勿勉强。
